Инициализировала новый локальный репозиторий с помощью команды git init. 
Подключила этот локальный репозиторий к удалённому репозиторию на GitHub, используя команду git remote add origin https://github.com/TashaStav/stepik_fullstack.git.
Создала файл index.html в локальном репозитории, добавила его в индекс с помощью git add .
Создала коммит с описанием изменений командой git commit -m "".
Отправила созданный коммит в удалённый репозиторий на GitHub с помощью команды git push origin main.
